Chapter 03

第一个工具

实现完整的 Tool Use 循环，让 Claude 能读取你的文件

到目前为止 Claude 只能说话。但 Claude Code 能读文件、执行命令——因为 API 支持 tool use。这一章我们实现完整的工具调用循环：Claude 决定要读一个文件 → 我们执行读取 → 把结果喂回去 → Claude 基于文件内容回答。

Tool Use 循环

先看全貌。整个流程是一个循环：

Tool Use 循环：Claude 可以多次调用工具，直到它认为已经收集够信息来回答

定义工具

工具定义就是一个 JSON Schema，告诉 Claude 这个工具叫什么、做什么、接受什么参数。在 Java SDK 中，我们用 Builder 模式构建：

import com.anthropic.core.JsonValue;
import com.anthropic.models.messages.Tool;
import java.util.List;
import java.util.Map;

var readFileTool = Tool.builder()
    .name("read_file")
    .description("Read the contents of a file at the given path. Returns the file content as text.")
    .inputSchema(Tool.InputSchema.builder()
        .putAdditionalProperty("type", JsonValue.from("object"))
        .putAdditionalProperty("properties", JsonValue.from(Map.of(
            "path", Map.of(
                "type", "string",
                "description", "The path to the file to read"
            )
        )))
        .putAdditionalProperty("required", JsonValue.from(List.of("path")))
        .build())
    .build();

Claude 不会直接执行这个工具——它只是看到这个定义，然后在需要时请求调用。真正的执行在我们的代码里。

执行工具

一个简单的方法，根据工具名执行对应操作。我们用 Jackson 的 JsonNode 来解析工具输入：

import com.fasterxml.jackson.databind.JsonNode;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;

private static String executeTool(String name, JsonNode input) {
    if (name.equals("read_file")) {
        try {
            var path = input.get("path").asText();
            return Files.readString(Path.of(path));
        } catch (IOException e) {
            return "Error: " + e.getMessage();
        }
    }
    return "Error: Unknown tool \"" + name + "\"";
}

Agent Loop

这是整章的核心——也是 AI Agent 的核心。一个 while(true) 循环，不断让 Claude 调用工具直到它决定停止：

private static void agentLoop(AnthropicClient client, String userMessage) {
    messages.add(MessageParam.ofUser(userMessage));

    while (true) {
        var response = client.messages().create(
            MessageCreateParams.builder()
                .model("claude-sonnet-4-20250514")
                .maxTokens(4096)
                .tools(List.of(readFileTool))
                .messages(messages)
                .build()
        );

        // 把 assistant 的完整回复加入历史
        messages.add(MessageParam.ofAssistant(response.content()));

        // 收集 tool_use 块
        var toolUseBlocks = response.content().stream()
            .filter(b -> b instanceof ContentBlock.ToolUse)
            .map(b -> (ContentBlock.ToolUse) b)
            .toList();

        // 如果没有工具调用，输出文本并结束
        if (toolUseBlocks.isEmpty()) {
            response.content().stream()
                .filter(b -> b instanceof ContentBlock.Text)
                .map(b -> (ContentBlock.Text) b)
                .forEach(t -> System.out.println(t.text()));
            break;
        }

        // 执行每个工具调用，收集结果
        var toolResults = new ArrayList<ContentBlockParam>();
        for (var toolUse : toolUseBlocks) {
            System.out.println("  [tool] " + toolUse.name() + "(" + toolUse.input() + ")");
            var result = executeTool(toolUse.name(), toolUse.input());
            toolResults.add(ContentBlockParam.ofToolResult(
                ToolResultBlockParam.builder()
                    .toolUseId(toolUse.id())
                    .content(result)
                    .build()
            ));
        }

        // 把工具结果作为 user 消息发回
        messages.add(MessageParam.ofUser(toolResults));
        // 继续循环 - Claude 会看到工具结果并决定下一步
    }
}

关键点解释

为什么工具结果放在 user 角色？

Anthropic API 的消息格式要求 user / assistant 严格交替。tool_result 是对 tool_use 的回复，逻辑上属于"我们给 Claude 的输入"，所以放在 user 侧。在 Java SDK 中用 MessageParam.ofUser(toolResults) 来构建。

while(true) 为什么安全？

因为如果 Claude 的响应里没有 ContentBlock.ToolUse 块，循环就 break 了。这就是 Agent Loop 的核心逻辑：循环直到 AI 决定停止调用工具。你也可以检查 response.stopReason()——它是 StopReason.TOOL_USE 时表示 Claude 还想继续，是 StopReason.END_TURN 时表示它说完了。

完整程序

把 REPL + Agent Loop + read_file 组装成完整的 Main.java：

package com.minicode;

import com.anthropic.client.AnthropicClient;
import com.anthropic.core.JsonValue;
import com.anthropic.models.messages.*;
import com.fasterxml.jackson.databind.JsonNode;

import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.*;

public class Main {
    private static final List<MessageParam> messages = new ArrayList<>();

    private static final Tool readFileTool = Tool.builder()
        .name("read_file")
        .description("Read the contents of a file at the given path")
        .inputSchema(Tool.InputSchema.builder()
            .putAdditionalProperty("type", JsonValue.from("object"))
            .putAdditionalProperty("properties", JsonValue.from(Map.of(
                "path", Map.of("type", "string", "description", "File path to read")
            )))
            .putAdditionalProperty("required", JsonValue.from(List.of("path")))
            .build())
        .build();

    private static String executeTool(String name, JsonNode input) {
        if (name.equals("read_file")) {
            try {
                return Files.readString(Path.of(input.get("path").asText()));
            } catch (IOException e) {
                return "Error: " + e.getMessage();
            }
        }
        return "Error: Unknown tool \"" + name + "\"";
    }

    private static void agentLoop(AnthropicClient client, String userMessage) {
        messages.add(MessageParam.ofUser(userMessage));

        while (true) {
            var response = client.messages().create(
                MessageCreateParams.builder()
                    .model("claude-sonnet-4-20250514")
                    .maxTokens(4096)
                    .tools(List.of(readFileTool))
                    .messages(messages)
                    .build()
            );

            messages.add(MessageParam.ofAssistant(response.content()));

            var toolUseBlocks = response.content().stream()
                .filter(b -> b instanceof ContentBlock.ToolUse)
                .map(b -> (ContentBlock.ToolUse) b)
                .toList();

            if (toolUseBlocks.isEmpty()) {
                response.content().stream()
                    .filter(b -> b instanceof ContentBlock.Text)
                    .map(b -> (ContentBlock.Text) b)
                    .forEach(t -> System.out.println(t.text()));
                break;
            }

            var toolResults = new ArrayList<ContentBlockParam>();
            for (var toolUse : toolUseBlocks) {
                System.out.println("  [tool] " + toolUse.name() + "(" + toolUse.input() + ")");
                var result = executeTool(toolUse.name(), toolUse.input());
                toolResults.add(ContentBlockParam.ofToolResult(
                    ToolResultBlockParam.builder()
                        .toolUseId(toolUse.id())
                        .content(result)
                        .build()
                ));
            }

            messages.add(MessageParam.ofUser(toolResults));
        }
    }

    public static void main(String[] args) {
        var client = AnthropicClient.builder()
            .apiKey(System.getenv("ANTHROPIC_API_KEY"))
            .build();

        var scanner = new Scanner(System.in);
        System.out.println("MiniCode v0.2 (type \"exit\" to quit)\n");

        while (true) {
            System.out.print("you> ");
            var input = scanner.nextLine().trim();
            if (input.equals("exit")) break;
            if (input.isEmpty()) continue;
            agentLoop(client, input);
            System.out.println();
        }
    }
}
